JavaScript Functions: Basic

functions documented

standInLine - find who's next in line

listVar - add a new food to our food list (Food, Quantity)

unshiftVar - add new a breakfast item to list, "run" will return new list and replace user input with the first item of the list ("omelete", "waffles", "french toast", "pancakes")

shiftVar - add new fruit to list, "run" will return new list and remove the first food in the list ("chicken", "sausage", "pork", "steak")

popVar - add new fruit to list, "run" will return new list and remove the fruit at the end of the list ("bananas", "apples", "limes", "strawberry")

pushVar - type a NBA team to have it pushed into Array

nthToLastLetter - will return the nth-to-last letter in each word (Can you figure out which letter?) (Minimum - 5 letters)

lastLetter - will return the last character in each word

nthLetter - will return the nth letter in each word (Can you figure out which letter?) (Minimum - 5 letters)

firstLetter - will return the first letter in each word (play around to create new word)

findLength - will return the character length of each input

constructString - will add favorite fruit to string "My favorite fruit is a/an... because it is delicious."

concatenatePlusEquals - will concatenate the first string (Name) to second existing string

concatenate - will concatenate the two inputs together (First box = adjective, Second box = noun)

multiDecimal - mutiplies decimals

decrement - subtracts one to existing number

increment - adds one to existing number
